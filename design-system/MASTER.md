# Design System — Master OAuth

Generated following [ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill) reasoning for a **cybersecurity / trust-authority auth portal**.

## Pattern
Trust & Authority + Soft Glass panel (single composition auth screens)

## Style
Soft UI Evolution / Glassmorphism (light)

## Colors
| Token | Value | Use |
|-------|-------|-----|
| Primary | `#0D7377` | Brand, CTA |
| Primary Deep | `#095456` | Gradient end |
| Accent | `#C27803` | Meta highlights |
| Background | `#EEF3F7` → `#D9E6EF` | Atmospheric mesh |
| Ink | `#122033` | Text |
| Muted | `#5B6B7C` | Supporting text |

## Typography
- Display: **Syne**
- Body: **IBM Plex Sans**

## Effects
- Soft glass panels + backdrop blur
- 200–300ms hover/focus transitions
- Orb float + rise entrance
- Respect `prefers-reduced-motion`

## Avoid
- AI purple/pink gradients
- Emoji-as-icons
- Harsh neon / OLED-only dark default
- Dense dashboard clutter on auth screens
